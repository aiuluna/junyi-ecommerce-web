购物流程

1. 注册并登陆网站
1 ) 新用户注册：点击首页顶部“快速注册”进入注册页面
输入手机号、短信验证码、密码，按照提示完成注册。
2 ) 老用户登陆：请在“用户登陆”页面输入登录用户名，密码进行登录。
2. 选购商品
1 ) 浏览要购买的商品，选择购买数量点击“立即购买”，进入下一环节。
2 ) 或者在购物车中凑单，选择好商品及数量后“去结算”，进入下一环节。
3. 提交订单
选择收货地址、使用优惠券、确认价格信息，确认无误后点击“提交订单”，生成订单，进入下一环节。
4. 订单支付
订单提交后，需在规定时间内完成订单支付，超时订单将自动取消关闭。
蚂蚁海购支持：微信支付、支付宝支付
5. 订单支付
订单支付完成后，蚂蚁海购将会在最短时间内为您办理清关与发货。
6. 确认收货
确认收到货物时，请您在订单详情页面或订单列表点击“确认收货”，完成本次订单。 如果您收到的货物有质量问题，可联系我们的客服人员帮您处理，亦可申请售后处理，详见售后服务说明。支付
