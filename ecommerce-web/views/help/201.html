退货政策

退换货原则
蚂蚁海购所售商品都是海外正品，生产标准都是按照商品生产国的相关政策及要求生产，和国内的生产标准有可能会不同，如果是因为标准不同而 产生的退换货，我们将不予受理。如符合以下情况的，我们可以按照国家三包政策，针对所售商品履行保修、换货和退货的义务。
允许退换货的条件
1 ) 收到货物后不超过7天的（以快递网站签收时间为准）。
2 ) 不影响二次销售。未开封，未使用，包装完好，物品状态良好。
3 ) 所附带的所有附件，礼品，收据，本商城独特辨识码都保持完好无损。
4 ) 商城在商品描述中和实际收到的商品不符的
不符合退换货的条件
1 ) 用户导致的诸如破损，质量问题，错发品种，错误尺寸和颜色，有效期失效，完全不符合物品信息中的自定义描述。
2 ) 商品描述中不允许退换的商品(产品质量问题除外)
3 ) 由于快递运输过程中的损坏，不能提供证明的。
4 ) 产品过敏性问题不属于质量问题。
特别提醒
1 ) 强烈建议用户在签收前，当着快递人员面仔细检查到货情况，如有问题，请务必拒签要求退回，或者当场拍照取证并获得快递人员签字认可。
2 ) 取证范围包括但不限于：
 - 包裹外表, 快递单号,包装上的独特标识等。
 - 打开包裹时刻的状态，包装状态，相应数量。
 - 有损坏物品细节。
 - 取证日期
