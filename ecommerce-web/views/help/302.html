运费说明

1 ) 提交订单时，系统会为您收货地点自动选择快递配送，邮费根据实际商品标示重量自动计算。

2 ) 具体运费计算规则：按照产品的毛重加包装重量，重量在500g以下（含500克）时，运费20元人民币；当重量超过500克时，每增加500g，运费增加16元人民币。（注：当重量不足500克时，以500克计）。

3 ) 当您参加蚂蚁海购发起的包邮活动时，对指定商品是不需要再额外支付快递费用哦！
