前言

2、服务条款的接受：

a 本协议所称的“用户”是指完全同意所有服务条款并完成注册程序的最终用户。

b 用户注册、使用蚂蚁海购网的行为即表示其已经完全同意并自愿接受服务条款的约束。

c 蚂蚁海购网通过其网站不定时发布的各项规则、公告、通知等为本协议不可分割的一部分。

3、服务条款的变更和修改
